# รายรับรายจ่าย Auto

โปรเจกต์บันทึกรายรับ-รายจ่ายแบบอัตโนมัติ โยนรูปสลิปโอนเงิน/ใบเสร็จเข้าไป ระบบจะ OCR
อ่านรูปแล้วแยกยอดเงิน วันที่ เวลา ร้านค้า และหมวดหมู่ให้เอง มี 2 ส่วน:

## 📱 `expense-tracker.html` — เว็บแอพ

เว็บแอพไฟล์เดียว เปิดในเบราว์เซอร์ได้เลยไม่ต้องติดตั้งอะไร ลากรูปสลิปมาวาง (เลือกได้หลายไฟล์
พร้อมกัน) แล้วแก้ไขข้อมูลก่อนบันทึกได้ ข้อมูลเก็บใน localStorage ของเบราว์เซอร์ มีสรุปยอด
แยกหมวดหมู่ และ export เป็น CSV ได้

**วิธีใช้**: เปิดไฟล์ `expense-tracker.html` ด้วยเบราว์เซอร์โดยตรง

## 🤖 `discord-bot/` — Discord Bot

บอท Discord ที่ให้ผลลัพธ์เดียวกันแต่ทำงานอัตโนมัติเต็มรูปแบบ — ทักส่งรูปสลิปหาบอท
บอทจะ OCR + บันทึกให้ทันที แล้วตอบกลับเป็นสรุปพร้อมปุ่มแก้ไข/สลับรายรับ-รายจ่าย/ลบ
มี slash command `/summary`, `/list`, `/export` (Excel/CSV), `/delete_last`

**วิธีติดตั้ง/รัน**: ดูรายละเอียดทั้งหมดใน [`discord-bot/README.md`](discord-bot/README.md)

## ที่มา

ทั้งสองส่วนใช้ logic การอ่านสลิป (`parser.js` / โค้ดในเว็บแอพ) เหมือนกัน: แก้ปัญหา Unicode
สระอำที่ Tesseract มักอ่านแยกส่วน, กันการเดายอดเงินผิดจากตัวเลขขยะที่ OCR อ่านผิด,
และมี image preprocessing (grayscale + ปรับคอนทราสต์) ช่วยให้อ่านสลิปพื้นหลังสีได้แม่นขึ้น
