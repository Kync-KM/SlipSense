# SlipSense

โยนรูปสลิปโอนเงินหรือใบเสร็จเข้าไป ระบบอ่านและบันทึกรายรับ-รายจ่ายให้อัตโนมัติ ไม่ต้องพิมพ์เอง

## Workflow

1. **รับรูป** — ผู้ใช้ส่งรูปสลิป/ใบเสร็จเข้าเว็บแอพ (ลากวาง) หรือทักหา Discord bot (แนบรูป)
2. **เตรียมรูป** — แปลงเป็นขาวดำและปรับคอนทราสต์ก่อน เพื่อให้ OCR อ่านตัวอักษรบนพื้นหลังสี
   ของสลิปธนาคารได้แม่นขึ้น
3. **OCR** — อ่านข้อความในรูปด้วย Tesseract (รองรับไทย+อังกฤษ) ได้ข้อความดิบออกมา
4. **Parse** — ตัวแยกข้อความ (`parser.js`) ดึงยอดเงิน วันที่ เวลา ชื่อร้าน/ผู้รับ และเดาหมวดหมู่
   จากคำสำคัญ พร้อมแก้ปัญหาที่ OCR ไทยมักเจอ เช่น สระ "ำ" ถูกอ่านแยกเป็นคนละตัวอักษร
5. **กันเดาผิด** — ถ้ายอดเงินอ่านไม่ออกจริงๆ (ตัวเลขเสียหายจน OCR อ่านเป็นขยะ) ระบบจะ**ไม่เดา**
   แต่ปล่อยว่างไว้พร้อมเตือนให้กรอกเอง กันไม่ให้ยอดผิดหลุดเข้าไปในสรุปเงินแบบเงียบๆ
6. **บันทึก + แก้ไขได้** — เว็บแอพให้ตรวจก่อนกดบันทึก ส่วนบอท Discord บันทึกอัตโนมัติทันที
   แล้วตอบกลับเป็นสรุปพร้อมปุ่มแก้ไข/สลับรายรับ-รายจ่าย/ลบ ถ้าอ่านผิดแก้ทีหลังได้เสมอ
7. **สรุปผล** — ดูยอดรวมรายรับ-รายจ่ายแยกตามเดือน/หมวดหมู่ และ export เป็น Excel หรือ CSV ได้

## ส่วนประกอบ

| ส่วน | คำอธิบาย |
|---|---|
| `expense-tracker.html` | เว็บแอพไฟล์เดียว เปิดในเบราว์เซอร์ได้เลย ข้อมูลเก็บใน localStorage |
| `discord-bot/` | บอท Discord ทำงานอัตโนมัติเต็มรูปแบบ มี slash command `/summary` `/list` `/export` `/delete_last` — ดูวิธีติดตั้งใน [`discord-bot/README.md`](discord-bot/README.md) |

ทั้งสองส่วนใช้ logic การอ่านสลิปชุดเดียวกัน (พอร์ตจาก `parser.js`) ผลลัพธ์จึงตรงกันไม่ว่าจะใช้ผ่านเว็บหรือบอท
